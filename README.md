# fintech

## How to run in IDE
* pip install pip-compile
* pip install -r requirements.txt
* Open investment.finviz_downloader and run it in IDE

## How to run in local host environment
* pip install pip-compile
* pip install -r requirements.txt
* python -m investment.finviz_downloader
